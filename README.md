## 演習問題の解答 
[exercises](./exercises) ディレクトリに章ごとの .md ファイルがあり、そこに演習問題の解答を記載しています。  
GitHub の数式のレンダリングの問題で見た目が少し分かりにくい部分（太字が太字になっていないなど）がありますが、ご了承ください。  
また、解答は一例に過ぎないものも多いので、ぜひより良い解答を考えてください。
